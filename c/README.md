#tlv

It provides TLV constructing and parsing functions.


What can it do
==============

1. Construct TLV struction as a tree;

2. Edit TLV tree;

3. Dump TLV tree into bytes buffer;

4. Load TLV tree from bytes buffer;

5. TLV tree traverse


How to compile it
=================

check Makefile and run GNU make.
